## 🚀 Próximos Pasos

### Para Contribuidores:
1. Clonar el repositorio:
```bash
git clone https://github.com/erickson558/earnapp-reviewer.git
cd earnapp-reviewer
```

2. Crear ambiente virtual:
```bash
python -m venv venv
.\venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

3. Instalar dependencias:
```bash
pip install -r requirements.txt
playwright install chromium
```

4. Ejecutar aplicación:
```bash
python main.py
```

### Para Nuevas Versiones:
Ver [GIT-WORKFLOW.md](GIT-WORKFLOW.md) para el proceso completo de versionamiento.

Resumen rápido:
```bash
# 1. Hacer cambios y commits
git add .
git commit -m "tipo: descripción"

# 2. Actualizar archivos de versión
# Editar: VERSION y config.json

# 3. Crear commit de versión

git commit -m "chore: bump version to vX.Y.Z"

# 4. Crear tag
git tag -a vX.Y.Z -m "Version X.Y.Z: descripción"

# 5. Pushear todo
git push origin main
git push origin vX.Y.Z
```
