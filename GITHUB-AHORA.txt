## Si Algo Falla: Pasos Individuales

### Paso 1: Verificar GitHub CLI
```powershell
gh --version
```

Si dice "gh not found":
1. Descarga de https://cli.github.com/
2. Instala
3. Abre terminal nueva

### Paso 2: Loguearse en GitHub
```powershell
gh auth status
```

Si dice "Logged out":
```powershell
gh auth login
# Sigue instrucciones interactivas
```

## 📦 Para Nueva Versión (cuando hagas cambios importantes)

```powershell
# 1. Cambiar versión
echo "1.0.1" > VERSION

# 2. Agregar cambio
git add VERSION

# 3. Commit
git commit -m "chore: Bump to 1.0.1"

# 4. Crear tag
git tag -a v1.0.1 -m "Version 1.0.1"

# 5. Push
git push origin main
git push origin v1.0.1

# GitHub Actions automáticamente crea la release!
```
